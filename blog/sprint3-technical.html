Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones):

Control Flow is the order in which the computer executes the code.
Loops are code that just loop until the desired outcome executes, where the loop stops.
An example for control flow is just plain instructions. We may be given a set of instructions that we must complete in order from what we were first told to the last thing that was said, and we just go about it.
For loops, an example that I can think of is how we try to work on a task. Some people like to try to solve it, take a break, re-evaluate, then repeat, and end up getting no where, sometimes. But, when it works out, we get our desired result, which ends the loop until we get stuck again and then the cycle continues.

Both can be accessed similarly with [] bracket notations for example, except Arrays use Index numbers because of where/what number the information is stored in that array. While objects use the property that we have used.

Explain what functions are and why they are useful:

Functions are a block of statements that perform a task. This is useful because if/when we want to execute a bunch of statements many times, we can just call the function which will runs those same set of statements till you get your desired result.
